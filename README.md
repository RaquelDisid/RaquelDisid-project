# Restaurante Sabor & Raíz

Página web para reservas y contacto del restaurante Sabor & Raíz.
Proyecto desarrollado como práctica de HTML, CSS y JavaScript.

## Tecnologías utilizadas
- HTML5 
- CSS3 con variables, flexbox, grid y media queries
- JavaScript (async/await, módulos, sessionStorage)

## Funcionalidades
- Formulario de reserva con validación en tiempo real
- Tres tipos de consulta: reserva estándar, evento privado y empleo
- Persistencia de datos con sessionStorage
- Envío simulado con fetch y mensajes de éxito/error
- Diseño responsive para móvil, tablet y escritorio
- Accesibilidad: navegación por teclado, lectores de pantalla
